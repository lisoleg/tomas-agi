## 滚动条深度修复（min-h-0 链 + 强化 CSS）

### 根因
flex 布局链中两个关键容器缺 min-h-0，导致 flex-1 子元素按内容撑高而非被父容器约束：
- App.tsx `main` (line 88): `flex-1 flex flex-col min-w-0` → **缺 min-h-0**
- ChatArea.tsx wrapper (line 64): `flex-1 flex flex-col min-w-0` → **缺 min-h-0**
- 结果：ChatArea 高度 = 内容高度（而非可用视口高度）→ MessageList 从不溢出 → 滚动条永不出现

### 修复
- App.tsx: `min-w-0 min-h-0`
- ChatArea.tsx: `min-w-0 min-h-0`
- index.css: 滚动条宽度 6→10px，不透明度 0.18→0.25（hover 0.45, active 0.6），加 !important 防 Tailwind reset，border 2px padding-box 制造轨道间距
- 新增全局 `[class*="overflow-y-auto"]` Firefox fallback

### 约束链
`h-screen` → `main(min-h-0)` → `ChatArea(flex-1+min-h-0)` → `MessageList(flex-1+min-h-0+overflow-y-auto)` ✅

### 设计哲学
- EML 容纳冲突，不自动覆盖旧知识
- 检测到冲突 → 展示对比 → 用户逐条决策 → 持久化记录
- 四选一: 📌保留旧的 / 🆕保留新的 / 🔀合并两者 / 👁️忽略

## 图谱可视化 UI 优化 (EMLGraphVisualization)
- **节点半径**: `8 + delta*20` → `Math.max(18, 14 + delta*35)`，确保圆足够大放文字
- **文字位置**: 圆外 (`d.radius + 12`) → 圆内居中 (`dominant-baseline: central, dy: 0`)
- **字体大小**: 固定 10px → 自适应半径 (`Math.max(8, min(radius * 0.55, 14))`)
- **文字截断**: 超长标签自动截断 + 省略号 (maxChars = radius / 4)
- **悬停交互**: 鼠标移到圆上 → 圆上方显示完整名称（amber 放大标签）+ tooltip 概念名高亮 + 圆圈金色描边
- **力导向参数**: link distance 60→100, charge -200→-400, collision padding +5→+8
- **缩放范围**: [0.1, 4] → [0.15, 8]，节点多时默认缩放 0.8 居中
- **画布高度**: DistillPanel 中 500 → 620
- **tooltip 样式**: amber 边框圆角 + 概念名 base 字体高亮 + δ/𝕀(X) 参数小字

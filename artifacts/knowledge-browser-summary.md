# 知识浏览器 + 添加到知识库 — 实现总结

## 1. 知识浏览器（📚 Tab）

蒸馏模式下，Token Bridge 加载图谱后，新增第三个 Tab「📚 知识浏览 (总数)」。

### 功能
- **统计概览**：V（概念数）/ E（关系数）/ K（知识条数）/ 𝕀̄ 均值，四格彩色卡片
- **分区切换**：全部 / 概念 / 关系，三个按钮切换视图
- **概念列表**：每行显示名称 + 𝕏（信息存在度），悬停出现 🗑️ 删除按钮
- **关系列表**：每行显示 src → dst + 类型 + 强度 w，悬停出现 🗑️ 删除按钮
- **删除逻辑**：
  - 删除概念 → 自动删除其所有关联关系（弹窗确认 + 告知影响多少条关系）
  - 删除关系 → 仅删除该条边
  - 删除后自动调用 `rebuildGraphAfterDelete` 重建 EML 图谱 → 自动加载回 Token Bridge → 同步更新可视化

### 涉及文件
- `src/api/distiller.ts` — 新增 `rebuildGraphAfterDelete()` 函数
- `src/components/DistillPanel.tsx` — 新增 `KnowledgeBrowser` 子组件 (192 行)

## 2. 添加到知识库按钮

蒸馏完成后（phase === 'done'），在操作区新增「📥 添加到知识库」按钮。

### 行为
- **Token Bridge 为空** → 直接将蒸馏结果加载到知识库
- **Token Bridge 已加载** → 触发合并预览（重叠/冲突检测），确认后合并
